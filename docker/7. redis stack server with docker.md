# Redis stack server安装with docker


## 1. 下载镜像	

* `redis/redis-stack`：包含Redis Stack服务器和RedisInsight可视化工具，适合本地开发（可通过RedisInsight直观查看数据）。
* `redis/redis-stack-server`：仅包含Redis Stack服务器，不含RedisInsight，适合生产环境部署。

```bash
 docker pull redis/redis-stack

 docker pull redis/redis-stack-server
```

## 2. 启动

```
docker run -d --name redis-stack -p 6379:6379 -e REDIS_ARGS="--requirepass abc123456" redis/redis-stack-server:latest

```

## 3. 进入命令行

```
docker exec -it redis-stack redis-cli

```
